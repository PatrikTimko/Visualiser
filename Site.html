ANTLR4 Denotational Semantics Visualizer

Description

This Visualiser tool is designed to parse and visualize simple programming language constructs based on a denotational semantic. It supports basic operations such as arithmetic calculations, conditional statements, sequences of actions, and variable assignments. Ideal for educational purposes, it provides a visual insight into the parsing process of programming languages.

Tutorial

  1. Input Code: Enter your code into the text area labeled 'Input'. The code should follow the defined grammar rules, supporting variable assignments, arithmetic operations, conditional statements, and sequences of actions.
  2. Parse: Click the 'Parse' button to process your input. The parser will interpret your code according to the custom grammar and begin the visualization process.
  3. View Results: After parsing, check the console for a detailed breakdown of the parsing process and a visual representation of the parse tree. For a summarized output, the 'Output' area will display a confirmation message.

Grammar Rules

The Visualiser supports a simple syntax for educational purposes. Here are the key grammar rules:

  • Variable Assignment: variable := expression
  • Conditional Statement: if condition then statement else statement
  • Sequence of Actions: seq statement; statement
  • Skip Action: skip
  • Expressions: Can include integers, variable identifiers, arithmetic operators (+, -, *, /), comparison operators (<, <=, >, >=, ==, !=), and nested expressions. If your expression contains comparison or arithmetic operation then use brackets:
    (expression operation expression)

Denotational Semantics rules

Assignment: Sds⟦x:=e⟧s = s⟦x ↦ Ɛ⟦e⟧s⟧

Sequence: Sds⟦S1;S2⟧ = (Sds⟦S2⟧ ᴑ Sds⟦S1⟧s)

Condition: Sds⟦if b then S1 else S2⟧ = cond(ß⟦b⟧, Sds⟦S1⟧, Sds⟦S2⟧)s
